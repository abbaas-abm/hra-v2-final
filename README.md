# HRA Transportation – Full-Stack Web Application  
**Live Site:** https://hratransportation.com  

This repository contains the complete source code for the custom moving and logistics platform built for **HRA Transportation (Pty) Ltd** in 2025.

### Project Overview (Delivered August – September 2025)

| Feature                                      | Status    | Description                                                                 |
|---------------------------------------------|-----------|-----------------------------------------------------------------------------|
| Responsive Website & Landing Page           | Completed | Branded, mobile-first design with clear CTAs                                 |
| Customer Portal                             | Completed | Register, place moving orders, track orders, view history                    |
| Driver Dashboard                            | Completed | Driver sign-up, profile (license + vehicle), accept jobs, job history        |
| Admin Dashboard (HRA)                       | Completed | Manage users, orders, commissions, basic analytics & reports                 |
| Secure Payment Integration                  | Completed | Success/failure redirect pages                                               |
| Role-based Authentication                   | Completed | Customer / Driver / Admin roles                                              |
| Commission Tracking                         | Completed | Automatic commission calculation per job for HRA                             |
| Custom T&C Pages                            | Completed | Delivery, Storage, Fridge Rental terms                                       |
| Contact Form + Email Delivery               | Completed | reCAPTCHA protected, emails sent to HRA                                      |
| Basic SEO & Performance Optimization        | Completed | Meta tags, clean URLs, fast loading, image optimization                      |
| SSL / Security                              | Completed | Full HTTPS, input sanitisation, rate limiting                                |
| Full Deployment                             | Completed | Hosted and running on client's domain                                        |

### Tech Stack

| Layer         | Technology                  |
|---------------|-----------------------------|
| Frontend      | React.js + Vite             |
| Backend       | Node.js + Express           |
| Database      | MongoDB (MongoDB Atlas)     |
| Authentication| JWT + bcrypt                |
| Payments      | PayFast / Ozow / custom gateway (as configured) |
| Hosting       | Client's own hosting / domain |
| Styling       | Tailwind CSS / custom CSS   |
